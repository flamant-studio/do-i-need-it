# Do I Need It? ⚠

An anti-impulse shopping tool. Paste a product URL. Answer five honest questions. Get a verdict.

**Live:** [doineedit.wtf](https://doineedit.wtf) *(or wherever it lands)*

---

## What it is

A satirical, single-flow web app that intercepts the moment between "I want this" and "I bought this." Five yes/no questions, a scoring system weighted toward NO, and four possible verdicts: HARD NO / NO / MAYBE FINE / YES (rare). Each NO verdict includes a philosopher quote and a 1-minute mindfulness quest as an alternative to buying. The YES screen is deliberately over the top.

It's less a utility and more a mirror held up to modern over-consumerism — in the spirit of Screen Time for shopping.

## Stack

- React (single component, no dependencies beyond React itself)
- Bebas Neue + IBM Plex Mono (Google Fonts)
- Pacifico (Google Fonts, YES screen only)
- No backend. No database. No auth. No tracking.
- Product identification via local URL parsing (no API calls)

## How the scoring works

Each question has a YES weight and a NO weight. Answering in ways that suggest impulse buying accumulates points. The verdict thresholds:

| Score | Verdict |
|-------|---------|
| ≥ 7 | HARD NO |
| 3–6 | NO |
| 1–2 | MAYBE FINE |
| ≤ 0 | YES (rare) |

Questions are drawn from a bank of 15. The first question is always "Is this replacing something that broke, wore out, or ran out?" (the key legitimacy filter) and the last is always "Will your life be worse without it?" Three middle questions are randomised each session.

## Quotes

26 quotes from Seneca, Epictetus, Thoreau, Nietzsche, Lao Tzu, Buddha, Zhuangzi, The Beatles, The Rolling Stones, Ariana Grande, Chuck Palahniuk, Vivienne Westwood, Art Buchwald, Sheryl Crow, and others. Randomised per verdict.

---

## Micro-quests

14 object-free 60-second mindfulness alternatives to buying. Randomised per verdict. Accessible via "GO ON A QUEST →" on the verdict screen. Completing the timer awards an insane number of karma points.
